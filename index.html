Bad Apples

For my final capstone project I developed an online implementation of one of my favorite games. Bad Apples is based on the bidding/bluffing game Skull, and with the site you can play against your friends online. I built separate client and server side apps and taught myself websockets to better implement the game.

Board Game Match

BGM is useful for people looking to play a board game. A user can implement the number of players and how difficult a game they want to play and the site retrieves the data, runs it through a simple algorithm and return the best games that match the criteria along with descriptions and an instructional video.

Hunters of the Week

Second project, focusing on creating a server API in Node with a Mongo Database, is a site based on the roleplaying game Monster of the Week. It allows users to build and maintain characters that they can then use to play the game.
