Board Game Match

BGM is useful for people looking to play a board game. A user can implement the number of players and how difficult a game they want to play and the site retrieves the data, runs it through a simple algorithm and return the best games that match the criteria along with descriptions and an instructional video.

Hunters of the Week

The focus of this student project was on creating a server API in Node with a Mongo Database. Based on the roleplaying game Monster of the Week, the site allows users to build and maintain characters that they can then use to play the game.
